# Ayesha & Basith Wedding Invitation

A mobile-friendly static wedding invitation website prepared for:

- **Bride:** Ayesha Rafa
- **Groom:** Abdul Basith
- **Date:** Monday, 28 December 2026
- **Time:** 11:30 AM
- **Venue:** Ansari Convention Centre, Perumpilavu

## Included
- Tap-to-open welcome screen
- Background music and music toggle
- Live countdown
- Google Maps venue button
- Bride and groom family details
- Invitation gallery
- WhatsApp RSVP button
- Responsive mobile and desktop layout

## Deploy to Vercel
1. Extract this ZIP file.
2. Upload the folder to a GitHub repository.
3. In Vercel, select **Add New → Project**.
4. Import the GitHub repository.
5. Set the project name to `basith-ayesha-invitation`.
6. Click **Deploy**.

Vercel will assign a public URL ending in `.vercel.app`. The requested project address is subject to availability.

## Replace or add photographs
Place new photos inside the `assets` folder. Then update the image paths in the gallery section of `index.html`.
